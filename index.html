⚙️ Firebase Ayarları

✅ Firebase projeniz bağlı: fclwebaraclari — Tüm özellikler aktif.
Farklı bir Firebase projesi bağlamak isterseniz aşağıdaki bilgileri güncelleyin. Firebase Console →
💡 Giriş yapan kullanıcı bu listedeyse otomatik Yönetici rolü alır.
📋 Firebase Güvenlik Kuralları (Firestore)
Firebase Console → Firestore → Rules bölümüne aşağıdaki kuralları yapıştırın:
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Araçlar: herkes okur, sadece admin yazar
    match /tools/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    // Sorular: herkes okur, giriş yapan yazar
    match /questions/{doc} {
      allow read: if true;
      allow create: if request.auth != null;
      allow update, delete: if request.auth != null;
      match /answers/{ans} {
        allow read: if true;
        allow write: if request.auth != null;
      }
    }
    // Duyurular: herkes okur, giriş yapan yazar
    match /announcements/{doc} {
      allow read: if true;
      allow write: if request.auth != null;
    }
    // Ziyaretçi defteri: herkes yazar, herkes okur
    match /guestbook/{doc} {
      allow read: if true;
      allow create: if true;
      allow update, delete: if request.auth != null;
    }
    // Kullanıcılar: kendi verisini okur/yazar
    match /users/{uid} {
      allow read: if true;
      allow write: if request.auth != null && request.auth.uid == uid;
    }
  }
}
🔐 Firebase Authentication Ayarları
Firebase Console → Authentication → Sign-in method → Email/Password ve Google sağlayıcılarını etkinleştirin.
📦 Firebase Storage Kuralları
rules_version = '2';
service firebase.storage {
  match /b/{bucket}/o {
    match /tool-logos/{file} {
      allow read: if true;
      allow write: if request.auth != null && request.resource.size < 2 * 1024 * 1024;
    }
  }
}
